En Git exiten Ramas o Branches ¿Cómo funcionan?

Las ramas son creadas para no afectar la rama maestra hsta que los cambios sean correctos.


Para cambiar la cabecera se puede usar git checkout master/(nombre_definido).

git commit -am "mensaje". Funciona para archivos que ya se han modificado nunca con nuevos.

git brach. Muestra que ramas existen actualmente.
